# Web Scraping Script for Laptops

This is a Python-based web scraping script that extracts laptop data from a test website. The script uses Selenium for web scraping and stores the data in a `laptop_jobs.json` file.

## Requirements

To run this script, you need the following:

- Python 3.x
- selenium for web scraping
- webdriver-manager to manage the Chrome WebDriver for Selenium
- A compatible version of Google Chrome installed on your machine.

## Setup Instructions

Follow these steps to set up the environment and run the script:

### 1. Clone the repository

Clone this repository to your local machine:

## install dependencies
pip install -r requirements.txt

## run the python file
python assignment.py

## outputfile
the output will be saved in laptop_jobs.json file
